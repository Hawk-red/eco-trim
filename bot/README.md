# EcoTrim Telegram Bot

Автоответчик для входящих сообщений в Telegram.

## Быстрый старт

```bash
cd bot
npm install
node index.js
```

## Настройка перед запуском

Открой `index.js` и замени две константы вверху файла:

| Константа | Что вставить |
|---|---|
| `ВСТАВЬ_ТОКЕН_СЮДА` | Токен от @BotFather |
| `ВСТАВЬ_СВОЙ_CHAT_ID` | Твой Telegram chat ID |

**Как получить токен:**
1. Напиши @BotFather в Telegram
2. Команда `/newbot`
3. Следуй инструкции — получишь токен вида `123456:ABC-DEF...`

**Как узнать свой chat ID:**
1. Напиши @userinfobot в Telegram
2. Он ответит твоим ID (число вида `123456789`)

## Что делает бот

- Отвечает на **любое** входящее сообщение карточкой с услугами и ценами
- Через 2 секунды присылает фото одного из проектов
- Уведомляет владельца о каждом новом клиенте

## Зависимости

- Node.js 16+
- `node-telegram-bot-api`
